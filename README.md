## Project Setup
Zuerst müssen alle Dependencies installiert werden:
```sh
npm install
```

### Compile VUE.JS + run ELECTRON (in dev mode)

```sh
npm run build:start
```

### Create Desktop Installer (production)

```sh
npm run make
```
